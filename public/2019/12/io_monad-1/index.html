严谨的程序显然不能接受运行到一半突然终止了，尤其是在一些服务端程序，这些程序通常需要一直运行提供服务，错误处理就显得格外重要。按照 Java 的习惯，我们只要在合适的地方加上 try catch 打印一行日志或显示系统错误 “假装” 已经处理了这些异常。
